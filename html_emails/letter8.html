Письмо отправлено 03 дек 2018 из https://lms.hse.ru для студентов дисциплины 11280 Единое окно студента (СПб)
Тема: Прием заявок для участия в конкурсе на повышенную стипендию

Открыт прием заявок для участия в конкурсе на получение повышенной государственной академической стипендии за особые достижения в учебной, научно-исследовательской, общественной, культурно-творческой, спортивной деятельности. 


Прием заявок осуществляется в системе LMS до 17:00  по московскому времени 25 декабря  через портфолио студента. Для участия в конкурсе необходимо разместить документы в портфолио, затем прикрепить их к заявке.

 Решением Ученого совета НИУ ВШЭ от 30 ноября 2018 г. по согласованию со Студенческим советом университета в Положение о стипендиях внесены изменения, касающиеся процедуры проведения конкурса и экспертизы заявок.

Основные изменения призваны сделать оценку конкурсных заявок в спортивном и общественных направлениях более прозрачной и консистентной, для чего будут сформированы экспертные комиссии, состоящие на 50% из сотрудников и на 50% из студентов университета. 
Кроме того, упрощается участие в конкурсе по научно-исследовательскому направлению для студентов магистратуры, поступивших в НИУ ВШЭ из других университетов.

Изменения, утвержденные на заседании ученого совета 30.11.2018 доступны по ссылке.

 Итоги конкурса будут подведены в конце января, повышенная стипендия будет назначена с 01.01.2019 по 30.06.2019 тем студентам, которым на этот же период будет назначена государственная академическая стипендия (стипендия НИУ ВШЭ за успехи в учебе – для студентов, обучающихся за счет НИУ ВШЭ).
